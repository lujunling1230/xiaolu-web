# Changelog

所有爱情公寓模块的变更记录。

## [1.1.0] - 2026-07-03

### Added

- 新增底部导航栏（Tab Bar）：消息、通讯录、发现、我
- 新增通讯录页面：展示7位AI室友及专属功能入口（如"找一菲谈心"、"给美嘉发消息"）
- 新增聊天记录本地存储：使用 localStorage 持久化每个角色的对话历史
- 新增消息时间戳：每条消息显示 HH:mm 格式时间
- 新增聊天列表动态时间：根据最后消息显示"今天"、"昨天"或 MM/DD
- 新增用户画像（userProfile）：默认标签"失眠"、"省钱"，AI回复更贴合人设
- 新增角色专属气泡主题色：7位角色各拥有独立配色和装饰SVG
- 新增"关于"弹窗彩蛋：点击右上角"?"查看爱情公寓介绍

### Changed

- 首页改为微信风格紧凑手机布局（320px 宽度）
- 底部导航栏仅显示在消息页和通讯录页，私聊页隐藏
- 右上角返回文案改为"回到 3602"
- 私聊不再触发群聊模式，仅一对一对话
- 陈美嘉人设 Prompt 优化：去除括号动作描写，改用纯口语台词风
- 吕子乔人设追加角色约束：禁止物化女性言论

## [1.0.0] - 2026-07-02

### Added

- 初始版本：爱情公寓·元宇宙客厅
- 7位经典角色（曾小贤、胡一菲、陈美嘉、吕子乔、关谷神奇、唐悠悠、张伟）
- 单聊模式：与指定角色一对一对话
- 群聊模式：关键词触发多角色接力吐槽
- 角色个性化气泡：线条小狗用户气泡 + 角色装饰SVG
- 音效反馈：拨号音、连接音、群聊提示音
- 入口集成到 MickeyLaunchpad

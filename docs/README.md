## 🚀 项目简介

这是我的第一个从 0 到 1 构建的数字作品集，包含两大核心板块：

| 板块 | 说明 |
|----|----|
| 🌲 森林疗愈室 | 专为 i 人设计的低能耗回血空间（呼吸引导 + 感恩日记） |
| 🧰 妙妙工具箱 | 覆盖生活 9 个维度的 AI 产品矩阵 |

本项目完全由 **React + TypeScript + Tailwind CSS** 构建，并通过结构化 Prompt 驱动 AI 完成开发。

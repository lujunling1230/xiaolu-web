# 时光博物馆 (Museum of Memories)

> 妙妙工具箱「魂」维度 -- 自我认知与个人成长叙事

---

## 产品简介

时光博物馆是妙妙工具箱中「魂」维度的核心产品，定位为个人作品集的压轴篇章。产品以"博物馆"为隐喻，将用户的人生片段划分为两个主题展厅 --「时代回响」与「荣耀之路」，通过复古胶片美学与粒子动画，营造时光沉淀的沉浸式浏览体验。

每一件展品都是一段记忆的切片：一首老歌、一部电影、一段网络文化碎片，或是一份引以为傲的荣誉。访客可以像逛博物馆一样，沿着时间轴线漫步，在泛黄的报纸卡片与金色光晕中，感受一段独特的成长叙事。

**访问路由**：`/toolbox/memories`

---

## 功能列表

### 展厅一：时代回响 (Hall of Echoes)

回顾伴随成长的年代文化印记，分为三个子类目：

| 子类目 | 内容 | 默认数量 |
|--------|------|----------|
| 时代BGM | 经典歌曲，附带网易云音乐播放 | 12 首 |
| 时代荧幕 | 影视作品 | 11 部 |
| 时代网络 | 互联网文化记忆（Flash游戏、贴吧、论坛） | 3 条 |

- 三个子类目通过顶部 Tab 自由切换
- 卡片以复古报纸风格呈现（深褐底色 + 暗金边框）
- BGM 类目支持迷你音乐播放器，点击即可播放网易云音乐
- 管理员可增删改查任意展品

### 展厅二：荣耀之路 (Hall of Glory)

记录个人成长中的高光时刻与重要荣誉：

| 默认荣誉 | 说明 |
|----------|------|
| 国家励志奖学金 | 学术类荣誉 |
| 优秀志愿者 | 社会实践类荣誉 |
| 数据库系统工程师证书 | 专业认证 |
| 网络安全证书 | 专业认证 |
| 英语四级证书 | 语言能力认证 |

- 垂直时间轴布局，按年份排列
- 支持展开/折叠复盘反思内容
- 重要荣誉附带金色光晕高光装饰
- 管理员可增删改查任意荣誉条目

### 全局功能

- **管理员认证**：密码保护，输入正确密码后解锁编辑权限
- **CRUD 操作**：支持展品的新增、编辑、删除
- **图片上传**：支持将图片转为 Base64 存储
- **确认删除**：删除操作前弹出确认对话框，防止误操作
- **数据持久化**：所有数据存储在浏览器 localStorage 中
- **数据迁移**：自动兼容旧版本数据，补全缺失字段
- **视觉特效**：尘埃飘落粒子动画 + 胶片显影入场动画

## 本地存储说明

所有展品数据存储在浏览器的 `localStorage` 中，共计 4 个 Key：

| Key | 数据类型 | 说明 |
|-----|----------|------|
| `museum_bgms` | `VintageCard[]` | 时代BGM 展品列表 |
| `museum_tvs` | `VintageCard[]` | 时代荧幕展品列表 |
| `museum_nets` | `VintageCard[]` | 时代网络展品列表 |
| `museum_honors` | `HonorItem[]` | 荣耀之路展品列表 |

**注意事项**：

- 数据仅存储在本地浏览器中，清除浏览器数据将导致数据丢失
- 不支持跨设备或多浏览器同步
- 图片以 Base64 编码存储，建议单张图片不超过 2MB
- localStorage 每个 Key 有 5MB 容量上限，大量高清图片可能超出限制
- 如需备份，建议定期导出数据（V1.1 计划支持 JSON 导出功能）

## 设计理念

### 视觉风格：复古胶片风

时光博物馆的设计灵感来源于老式胶片相机与泛黄报纸。整体采用深褐色系为主色调，搭配暗金色点缀，营造出时光沉淀的厚重感与温暖感。

**核心色板**：

| 色值 | 名称 | 用途 |
|------|------|------|
| `#5c4033` | VINTAGE_DEEP | 页面主背景色 |
| `#b08d57` | GOLD | 暗金边框、按钮、高亮元素 |
| `#8B6B4F` | VINTAGE_BROWN | 辅助色，文字装饰 |
| `#3a2a1a` | 卡片底色 | 复古报纸卡片背景 |
| `#f5e6d3` | 文字主色 | 泛黄纸张色正文 |

### 字体系统

- **主字体**：Noto Serif SC（思源宋体），展现印刷品的质感
- **备选字体**：Georgia, 'Times New Roman', serif
- **加载方式**：Google Fonts CDN

### 动画语言

- **胶片显影效果**：页面加载时，内容从模糊到清晰，模拟胶片冲洗过程
- **尘埃飘落粒子**：全屏 CSS 粒子动画，模拟老照片的尘埃与划痕
- **卡片过渡动画**：Tab 切换、复盘展开均使用 Framer Motion 的流畅过渡
- **金色光晕**：重要荣誉的高光装饰，通过 CSS 渐变 + 动画实现呼吸效果

### 产品哲学

> 每一段记忆都值得被陈列，每一份荣誉都值得被复盘。

时光博物馆不只是一个展示工具，更是一面镜子 -- 让用户在对过去的回望中，看清自己的成长轨迹，找到继续前行的力量。

## 快速开始

1. 访问 `/toolbox/memories` 进入时光博物馆
2. 浏览「时代回响」与「荣耀之路」两个展厅
3. 如需管理展品，点击页面顶部管理入口，输入密码 `ling` 解锁编辑权限
4. 管理员模式下，可自由添加、编辑、删除展品

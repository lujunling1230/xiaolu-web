# 通关清单（Quest Log）-- 妙妙工具箱

> RPG 风格游戏化待办清单 | 妙妙工具箱「行」维度

---

## 产品简介

通关清单是一个将待办事项转化为 RPG 冒险任务的游戏化工具。它把枯燥的 To-Do 变成"打怪升级"，通过 XP 经验值系统、Lv 等级、智能拆解弹窗、粒子爆炸完成动画和金币音效，让执行力变得有趣。

**核心理念**：把人生变成一场 RPG。

---

## 功能列表

### 任务管理

- **添加任务**：底部固定输入框，支持回车提交，自动弹出智能拆解弹窗
- **完成任务**：点击"完成"按钮，触发粒子爆炸动画 + 金币音效 + XP 奖励
- **编辑任务**：模态框修改任务描述和挑战等级，需管理员验证
- **删除任务**：管理员保护，需验证身份后删除

### 智能拆解

新增任务时自动弹出三种风险收益选项：

- **先做 5 分钟**（easy / +5 XP）：启动 300 秒倒计时，归零自动完成，标题追加"（5分钟挑战）"
- **只要 60 分**（normal / +60 XP）：标为及格线，标题追加"（及格线）"，降低心理负担
- **直接挑战**（hard / +100 XP）：满分完成，原样添加，完整奖励

### XP 经验值系统

- **等级机制**：每 100 XP 升一级，从 Lv.1 开始
- **霓虹进度条**：粉 -> 黄 -> 绿渐变填充，Spring 动画平滑过渡，发光外阴影
- **难度奖励**：easy +5 XP / normal +60 XP / hard +100 XP
- **数据持久化**：localStorage `quest_log_xp` 存储

### 视觉反馈

- **粒子爆炸**：完成时 14 个彩色粒子四散飞出（霓虹粉/电光蓝/柠檬黄/绿/紫），0.6s 动画
- **金币音效**：Web Audio API 合成 B5(988Hz) -> E6(1319Hz) 升频方波，模拟经典金币收集音
- **任务退出**：完成/删除时滑出 + 缩小动画
- **5分钟倒计时**：黄色高亮显示 MM:SS 格式倒计时

### 管理功能

- **管理员保护**：删除/编辑操作需通过 AdminGuard 验证
- **管理员入口**：右下角浮动圆形按钮（未登录显示锁图标，已登录显示齿轮图标）

## 本地存储说明

| Key | 类型 | 说明 |
|-----|------|------|
| `quest_log_data` | JSON Array | 任务列表（Quest[]），包含所有进行中和已完成的任务 |
| `quest_log_xp` | String | 累计经验值，存储为字符串数字（如 "350"），每 100 XP 升一级 |

**数据格式：**

```typescript
// quest_log_data
interface Quest {
  id: string;          // UUID
  text: string;        // 任务描述
  difficulty: "easy" | "normal" | "hard";
  completed: boolean;
  countdown?: number;  // "5分钟挑战"倒计时剩余秒数
}

// quest_log_xp
// 示例："350" 表示 350 XP，对应 Lv.4（350/100 + 1）
```

**隐私说明**：所有数据存储在浏览器本地 localStorage，不会上传至任何服务器。清除浏览器数据将导致所有记录丢失。

## 设计理念

### 视觉风格

- **暗黑 RPG 主题**：深色背景（深蓝-黑渐变），模拟游戏界面氛围
- **霓虹色渐变**：粉色 -> 黄色 -> 绿色渐变 XP 进度条，发光外阴影
- **难度色彩编码**：绿色(easy) / 黄色(normal) / 红色(hard) 三色体系
- **毛玻璃层次**：底部输入栏和弹窗遮罩使用 backdrop-filter 模糊
- **卡片式列表**：半透明白色背景 + 微弱边框，hover 高亮

### 交互原则

- **即时爽感**：完成任务的粒子爆炸 + 金币音效，提供即时正向反馈
- **降低阻力**：智能拆解弹窗，将"难以启动"的任务转化为可执行的小步骤
- **游戏化语言**：难度标签使用"先做 5 分钟/只要 60 分/直接挑战"等游戏化表述
- **风险收益选择**：每种难度对应不同的 XP 收益，让用户自行权衡

### 无障碍

- 所有交互元素支持键盘操作
- 不支持 Web Audio 时音效静默失败
- 不支持 `crypto.randomUUID` 时降级为时间戳 + 随机字符串

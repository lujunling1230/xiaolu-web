# 爱情公寓模块

> 把《爱情公寓》搬进屏幕——七位经典角色全天候在线，陪你吐槽、谈心、发牢骚。

## 功能概览

| 功能 | 说明 |
|------|------|
| 消息列表 | 微信风格的会话列表，展示群聊和7位角色私聊入口 |
| 通讯录 | 查看所有AI室友，点击即可发起聊天或查看专属功能 |
| 单聊 | 与指定角色一对一对话，支持历史上下文和本地存储 |
| 群聊 | 关键词触发多人群聊，角色接力吐槽 |
| 彩蛋 | 点击右上角"?"查看关于弹窗 |

## 如何使用

1. 从 MickeyLaunchpad 点击「爱情公寓」入口进入
2. **消息页**：查看会话列表，点击「公寓群聊」发起群聊，或点击角色头像进入私聊
3. **通讯录**：点击底部 Tab「通讯录」进入，查看所有AI室友及专属功能入口（如"找一菲谈心"、"给美嘉发消息"），点击即可发起聊天
4. 在聊天界面输入文字，AI角色会根据人设回复
5. 群聊模式：输入特定关键词（如"加班"、"没钱"）触发多人群聊接力
6. 点击左上角「回到 3602」返回列表

## 技术栈

- React + TypeScript + Vite
- Framer Motion 动画
- 本地存储（localStorage）持久化聊天记录
- 可选 AI 接口（deepseek-r1）

## 文件结构


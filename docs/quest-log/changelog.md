# CHANGELOG -- 通关清单（Quest Log）

---

## V1.0 (2026-07-03) -- 首次发布

### 新增功能

#### 任务管理

- 底部固定输入栏（毛玻璃透明背景），支持文本输入和回车快捷提交
- "添加"按钮，输入为空时 disabled 状态
- 任务列表以卡片式布局展示，按添加顺序排列
- 每个任务项包含：难度标签（彩色边框）、任务文案、操作按钮组
- 预设 3 条默认任务（读完 10 页书 / 搞定周报初稿 / 喝一杯水），其中 1 条已完成

#### 智能拆解弹窗

- 新增任务时自动弹出深色遮罩弹窗（spring 动画弹出）
- 任务标题黄色高亮显示在弹窗中
- 三种风险收益选项：
  - 先做 5 分钟（easy / +5 XP）：启动 300 秒倒计时，标题追加"（5分钟挑战）"，归零自动完成
  - 只要 60 分（normal / +60 XP）：标题追加"（及格线）"，标签化降低心理负担
  - 直接挑战（hard / +100 XP）：满分完成，原样添加
- 每个选项含图标 + 标签 + 描述 + XP 标签（带对应颜色边框和背景）
- 各选项 hover 时对应颜色边框发光 + 右移效果
- 支持点击遮罩或关闭按钮关闭弹窗

#### 5分钟挑战倒计时

- easy 模式任务自动启动 300 秒倒计时
- 倒计时以黄色 MM:SS 格式显示在任务文案旁
- 每秒递减，归零时自动触发完成逻辑（粒子爆炸 + 金币音效 + XP 奖励）
- 倒计时状态持久化至 localStorage，刷新页面后恢复

#### XP 经验值系统

- 每 100 XP 升一级，从 Lv.1 开始
- 等级显示：`Lv.X` 黄色标签（带半透明背景和边框）
- 霓虹渐变进度条：粉 -> 黄 -> 绿三色渐变，发光外阴影
- Spring 动画平滑过渡（stiffness: 120, damping: 14）
- 进度条显示当前经验值 / 100 XP
- 完成 easy 任务 +5 XP，normal 任务 +60 XP，hard 任务 +100 XP
- XP 数据持久化至 localStorage（key: `quest_log_xp`）

#### 视觉反馈

- **粒子爆炸**：完成时生成 14 个彩色粒子（霓虹粉/电光蓝/柠檬黄/绿/紫），从任务中心四散飞出，0.6s easeOut 动画（opacity 1->0, scale 1->0.2）
- **金币音效**：Web Audio API 合成 B5(988Hz) -> E6(1319Hz) 升频方波，两段间隔 80ms，各持续 180ms，音量 0.18 指数衰减
- **任务退出动画**：完成/删除时任务项滑出右侧 + 缩小消失（x: 60, scale: 0.8, opacity: 0, 0.35s）
- **按钮动效**：完成按钮 hover 放大 1.08 + 绿色发光，添加按钮 hover 放大 1.05

#### 编辑任务模态框

- 点击任务项编辑按钮（铅笔图标）打开模态框
- 模态框表单：任务描述输入框（自动聚焦）+ 挑战等级选择（三个按钮）
- 难度按钮：含图标 + 标签 + XP 值，选中态显示对应颜色边框发光 + 背景
- 取消/保存按钮，保存需管理员验证
- 保存后模态框关闭，列表项实时更新

#### 管理功能

- 管理员保护：删除/编辑操作需通过 AdminGuard 验证
- 右下角浮动圆形管理员入口按钮（44x44px，毛玻璃背景）
- 未登录：锁图标 + 半透明白色背景
- 已登录：齿轮图标 + 半透明绿色背景
- 点击触发 `verifyAdmin` 验证

#### 页面框架

- 顶部导航栏：左侧"回到妙妙工具箱"链接（hover 变黄 + 左移），右侧"Quest Log"标签
- 玩家状态区：XP 进度条 + 统计 pill（进行中数量 / 已通关数量）
- 标题区：大号白色标题"通关清单" + 副标题"把人生变成一场 RPG。"
- 空状态：游戏手柄图标 + 引导文案

### 技术实现

- 技术栈：React 18 + TypeScript + Framer Motion + Web Audio API
- 样式方案：组件级内联 `<style>` 标签，所有 CSS 类名使用 `quest-` 前缀
- 数据持久化：localStorage（`quest_log_data` / `quest_log_xp`），通过 `legacyLoad`/`legacySave` 工具函数
- ID 生成：优先使用 `crypto.randomUUID()`，不支持时降级为 `时间戳-随机字符串`
- 动画方案：Framer Motion（粒子爆炸、弹窗、任务退出、进度条） + AnimatePresence（列表动画）
- 响应式布局：max-width 720px 居中，移动端适配（弹窗描述文案隐藏、编辑难度按钮纵向排列）
- 音效容错：Web Audio API 不可用时静默失败，不影响核心功能
- 管理员保护：通过 `useAdminGuard` hook 实现操作级权限控制

### 设计风格

- 暗黑 RPG 主题：深色背景（深蓝-黑 radial-gradient 渐变），模拟游戏界面氛围
- 霓虹色渐变：XP 进度条使用粉 -> 黄 -> 绿三色渐变，带发光外阴影
- 难度色彩编码：easy 绿色 / normal 黄色 / hard 红色三色体系，贯穿标签、弹窗选项、按钮
- 毛玻璃层次：底部输入栏使用 backdrop-filter 毛玻璃透明背景，弹窗遮罩 4px 模糊
- 卡片式列表：rgba(255,255,255,0.04) 半透明背景 + 微弱边框，hover 高亮
- 游戏化语言：所有文案使用 RPG 风格表述（"先做 5 分钟"/"只要 60 分"/"直接挑战"/"已通关"）
- 字体：Noto Sans SC 为主，等宽数字字体用于倒计时显示

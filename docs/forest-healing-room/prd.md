## 一、产品背景与定位

### 1.1 背景

在快节奏的数字生活中，用户普遍面临焦虑、注意力涣散、情绪疲惫等问题。市面上的冥想/正念应用多为独立 App，下载门槛高、数据孤岛严重。妙妙工具箱作为一站式轻量工具集，需要一个内嵌的身心调节模块，帮助用户在碎片化时间内完成快速情绪复位。

### 1.2 定位

森林疗愈室是妙妙工具箱「身」维度的核心工具，定位为**沉浸式线上心理疗愈空间**。它通过森林自然意象包裹四个功能模块，为用户提供一个低门槛、零压力的自我关怀入口。

### 1.3 产品价值主张

- **低门槛**：无需下载额外 App，浏览器即开即用
- **零社交压力**：所有数据存储在本地 localStorage，隐私绝对安全
- **感官疗愈**：从视觉（森林背景/纸张质感）、听觉（白噪音环境音）、触觉（动画反馈）三个维度构建沉浸体验
- **持续陪伴**：感恩日记的 12 月主题设计和冥想累计时长记录，鼓励长期使用

### 3.2 模块详情

#### 3.2.1 导航框架（HealingRoomPage）

- **桌面端**：左侧 220px 固定侧边栏（backdrop-filter 毛玻璃 + 半透明深色背景），包含 4 个导航项，每项有中文标签、英文副标题、SVG 图标
- **选中态**：左侧绿色竖线指示器（Framer Motion layoutId 平滑过渡），背景色突出
- **移动端**：顶部 56px 固定栏 + 汉堡菜单按钮，点击展开下拉导航面板（AnimatePresence 动画）
- **内容区**：右侧 Flex 居中卡片（max-width 700px），模块切换带淡入淡出动画

#### 3.2.2 感恩日记（GratitudeJournal）

**三页交互模型：**

- **封面页**：水彩手绘背景图 + 柔光遮罩 + 漂浮光斑 + 中英文标题。点击任意处触发 3D 翻书动画进入目录
- **目录页**：4 列网格（移动端 3 列），12 个月度卡片。当前月份高亮边框 + 发光，已有日记的月份右上角绿点标记。hover 上浮，点击进入对应月份日记页
- **日记页**：
  - 顶部：返回目录按钮 + 月度主题药丸
  - 月度引导语：带左侧绿色边框的引用块，显示当月主题文案
  - 左栏：日期显示、天气选择器（4 种 SVG 图标）、心情色环（6 色可选，选中放大 + 白边 + 发光）
  - 右栏：书写文本域，纸张质感背景
  - 保存区：便签贴纸风格按钮（旋转 -2deg），保存后显示反馈文案 2 秒
  - 底部：本月历史记录列表，带时间轴竖线 + 圆点
  - 右下角：固定"合上书"按钮，返回封面

**体验细节：**
- 翻书动画使用 3D rotateY 变换，含脊柱阴影
- 内页使用手造纸质感（横线纹理 + 水彩晕染背景）
- 日夜模式自适应配色
- 支持 `prefers-reduced-motion` 减少动效

#### 3.2.3 呼吸引导（BreathingGuide）

**三种呼吸模式：**

| 模式 | 节奏 | 总时长 | 颜色 | 适用场景 |
|------|------|--------|------|----------|
| 方形呼吸 | 4-4-4-4（含空肺屏息） | 16s/循环 | 蓝色 #5B9BD5 | 焦虑/压力大 |
| 478 呼吸法 | 4-7-8 | 19s/循环 | 紫色 #7E57C2 | 疲惫/失眠 |
| 能量呼吸法 | 3-1-2 | 6s/循环 | 橙色 #FF8A65 | 能量不足 |

**动画引擎：**
- 使用 rAF（requestAnimationFrame）驱动，非 CSS 动画
- 光球缩放：吸气 0.76 -> 1.14，呼气 1.14 -> 0.76
- SVG 圆弧进度：吸气填充，屏息保持满，呼气消退
- 核心修复：`lastSecond = -1` 强制首帧更新，解决阶段切换时倒计时跳跃 Bug

**控制：**
- 开始/暂停/继续三态按钮
- 重置按钮
- 节奏指示器：圆点 + 阶段标签，当前阶段高亮
- 循环计数显示

#### 3.2.4 冥想空间（MeditationTimer）

**设置阶段：**
- 环境音选择：4 格网格卡片，支持选中高亮 + 1 个 Coming Soon 占位卡
- 时长选择：4 个 pill 按钮，等宽分布
- 语音引导：iOS 风格 Toggle 开关
- 开始按钮：根据语音引导开关显示不同文案

**冥想阶段：**
- 当前环境音标签显示
- SVG 进度环（r=92，strokeWidth=4），Framer Motion 动画
- 大号数字倒计时（等宽数字字体）
- 结束按钮（提前终止）

**完成阶段：**
- 三层涟漪扩散动画
- 完成文案 + 冥想时长
- "再来一次"按钮
- 累计时长自动累加到 localStorage

**Web Audio 环境音：**
- 使用 OscillatorNode + GainNode 实时合成
- 海浪：低频 LFO 调制脉动
- 森林：多频三角波叠加 + 带通滤波
- 篝火：低频锯齿波 + 方波 crackle
- 音乐：C4-E4-G4 正弦波循环

#### 3.2.5 留言板（MessageBoard）

**表单区：**
- 一体化表单框（淡绿背景）
- 昵称输入框（可选，20 字上限，留空默认"匿名访客"）
- 星级评分：5 星可点击交互，hover 预览
- 留言文本域（200 字上限，带实时字数统计）
- 发布按钮 + 成功反馈动画

**留言列表：**
- 2 列网格（移动端 1 列）
- 4 条种子评价（预置破冰内容）
- 用户留言左侧绿色强调边
- 新留言带"新"标签
- 随机头像分配（8 种植物/动物图标池）
- 卡片 hover 上浮效果

## 四、数据模型

### 4.1 感恩日记（localStorage key: `gratitude_entries` + `gratitude_drafts`）

```typescript
interface GratitudeEntry {
  id: string;        // 时间戳 ID
  date: string;      // YYYY-MM-DD
  month: number;     // 1-12
  mood: string;      // 天气 key: "sun" | "cloud" | "rain" | "moon"
  color: string;     // 心情色 key: "joy" | "calm" | "cool" | "tender" | "soulful" | "mellow"
  content: string;   // 日记正文
  time?: string;     // HH:mm，保存时自动填入
}

interface Draft {
  weather: string;
  color: string;
  content: string;
}
```

### 4.2 冥想空间（localStorage key: `meditation_total`）

```typescript
// 累计冥想秒数，存储为字符串数字
type MeditationTotal = string; // "3600" 表示 3600 秒
```

### 4.3 留言板

- 无持久化存储，页面刷新后留言重置为种子数据
- 运行时状态由 React useState 管理

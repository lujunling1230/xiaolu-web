# 解压馆 (Stress Relief Room) -- CHANGELOG

## V1.0 (2026-07-03) -- 首次发布

### 新增功能

- **游戏选择卡片**：三张横向卡片（无限捏泡泡、禅意切割、重力涂鸦），带渐变图标、名称、描述、开始按钮
- **卡片动画**：hover上浮6px+阴影增强+边框粉色，移动端纵向堆叠，桌面端横向排列
- **无限捏泡泡**：6x6=36个SVG彩色气泡网格，点击爆裂效果
  - 气泡爆裂：scale 1->0 + opacity 1->0，持续250ms
  - 星星粒子：6个"✦"字符向随机方向飞散旋转180度，持续600ms
  - 啵声音效：Web Audio sine波形，720Hz->220Hz，持续100ms
  - 重生机制：600ms后原地重生，随机换色
  - 防重复点击：已爆裂气泡不响应
- **禅意切割**：5个彩色方块画布，拖拽划线切割
  - 交互：PointerEvent统一触控/鼠标，白色折线实时显示轨迹
  - 切割判定：取轨迹中落入方块的首尾两点作为切割线
  - 分裂动画：clipPath分上下两半，上半向左上飞散（-6, -50, rotate -14），下半向右下飞散（8, 80, rotate 18），持续700ms
  - 重生：动画完成后方块在随机位置重生，随机换色
  - 方块装饰：圆角16px，内阴影高光，轻微上下浮动
- **重力涂鸦**：Canvas画线生成圆点，受重力下坠堆积
  - 物理引擎：requestAnimationFrame驱动，60fps模拟
  - 重力加速度：0.35px/帧
  - 圆点属性：位置(x,y)、速度(vx,vy)、半径(4-7px)、颜色、静止标记
  - 堆积检测：高度图按8px列宽划分，每列记录最高堆积点
  - 插值生成：两点间每4px生成一个圆点，保证画线连续
  - 性能控制：总量限制600个，超出时FIFO删除最早100个
  - 高清适配：devicePixelRatio缩放Canvas
  - 清空按钮：右下角垃圾桶图标，点击清空所有圆点+重置高度图，hover放大1.1倍
- **游戏切换**：游戏内"换一个"按钮返回卡片选择页，AnimatePresence mode="wait"平滑过渡
- **马卡龙色板**：6色全局色板（#ffb6c1/#b6d5e8/#b8e0d2/#d4c5e2/#f5e1b8/#f6c6c6），气泡/方块/圆点/粒子统一取色

### 技术实现

- **框架**：React 18 + TypeScript + Framer Motion
- **SVG渲染**：Pop It气泡网格、星星粒子、切割轨迹折线
- **Canvas 2D渲染**：Gravity Doodle圆点绘制，rAF循环，dpr适配
- **PointerEvent**：Zen Cut和Gravity Doodle的拖拽交互，统一触控和鼠标
- **Web Audio API**：sine波形OscillatorNode + GainNode，延迟创建AudioContext
- **CSS clipPath**：polygon()动态切割方块，实现分裂效果
- **AnimatePresence**：卡片<->游戏视图切换，mode="wait"确保退出动画完成
- **motion.div**：气泡爆裂、方块飞散、粒子扩散、卡片入场动画
- **高度图算法**：按列划分+相邻列更新，模拟堆积斜面效果
- **性能优化**：圆点FIFO限制（600上限），SVG节点数可控（36气泡+36粒子），DOM节点数可控（5方块+10碎片）

### 设计风格

- **色调**：粉色渐变背景（`radial-gradient`），马卡龙6色色板，低饱和度高明度
- **字体**：Noto Sans SC，标题粗体800，正文常规
- **材质**：白色卡片+圆角18px+柔和阴影，内阴影高光模拟立体感
- **构图**：居中宽列布局（max-width: 860px），卡片横向滚动/纵向堆叠
- **氛围**：甜美、柔和、无压力，马卡龙甜品店质感
- **交互**：直觉化点击/拖拽，无教程无引导，即点即玩

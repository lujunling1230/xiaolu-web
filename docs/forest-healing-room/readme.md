# 森林疗愈室 -- 妙妙工具箱

> 沉浸式线上心理疗愈空间 | 妙妙工具箱「身」维度

---

## 产品简介

森林疗愈室是一个基于浏览器的轻量级心理疗愈工具，集成在妙妙工具箱中。它以森林自然意象为视觉主题，提供四个核心疗愈模块，帮助用户在碎片化时间内完成快速情绪复位和自我关怀。

**设计理念**：低门槛、零压力、隐私安全、感官沉浸。

---

## 功能列表

### 感恩日记

以实体绘本为交互隐喻，提供完整的日记书写体验：

- **封面页**：手绘水彩风格封面，光斑漂移微动效，点击翻书进入目录
- **目录页**：12 个月度篇章网格，当前月份高亮，历史月份标记
- **日记页**：天气选择器 + 心情色环 + 书写文本域 + 月度引导语
- **历史记录**：本月所有日记按时间轴展示，支持删除
- **草稿自动保存**：离开页面前自动持久化

### 呼吸引导

基于科学呼吸法的交互式呼吸训练工具：

- **方形呼吸**（4-4-4-4）：适合焦虑/压力大时使用，4 阶段完整方形循环
- **478 呼吸法**（4-7-8）：适合疲惫/失眠时使用，延长呼气促进深度放松
- **能量呼吸法**（3-1-2）：适合能量不足时使用，快速节奏唤醒活力
- **实时动画**：rAF 驱动的光球缩放 + SVG 圆弧进度 + 数字倒计时
- **节奏指示器**：圆点 + 阶段标签，实时显示当前呼吸阶段

### 冥想空间

带环境音的正念冥想计时器：

- **环境音选择**：海浪 / 森林 / 篝火 / 音乐（Web Audio API 实时合成）
- **时长选择**：1 / 3 / 5 / 10 分钟四档
- **语音引导开关**：开启后增加引导提示
- **进度环**：SVG 圆环进度 + 剩余时间显示
- **完成动画**：三层涟漪扩散动画
- **累计记录**：冥想总时长持久化记录

### 留言板

轻量级社区互动模块：

- **种子评价**：4 条预设留言，破冰引导
- **用户留言**：昵称（可选）+ 星级评分 + 留言文本域（200 字上限）
- **实时展示**：新留言插入列表顶部，带"新"标签
- **随机头像**：8 种植物/动物图标池

## 本地存储说明

| Key | 类型 | 说明 |
|-----|------|------|
| `gratitude_entries` | JSON Array | 感恩日记条目列表，每次保存追加新记录 |
| `gratitude_drafts` | JSON Object | 感恩日记草稿，按月份索引，离开页面前自动保存 |
| `meditation_total` | String | 冥想累计总秒数，完成冥想后累加 |

**隐私说明**：所有数据存储在浏览器本地 localStorage，不会上传至任何服务器。清除浏览器数据将导致所有记录丢失。

## 设计理念

### 视觉风格

- **日系文艺风**：以"森林"为核心意象，低饱和绿色系为主色调
- **纸张质感**：内页模拟手造纸纹理（横线 + 纤维噪点），暖白/奶油色底
- **毛玻璃层次**：导航栏和内容卡片使用 backdrop-filter 毛玻璃效果，营造景深感
- **水彩晕染**：多层径向渐变叠加，模拟森林光晕透入纸面的温润感
- **细线 SVG 图标**：所有图标使用 SVG 细线风格，视觉统一克制

### 交互原则

- **呼吸感动画**：所有过渡使用 0.3-0.6s 的 easeOut 缓动，营造"缓慢而温柔"的节奏
- **触觉反馈**：按钮 hover 缩放/上浮，保存成功文案淡出，建立即时反馈闭环
- **3D 翻书隐喻**：感恩日记使用 rotateY 3D 变换 + 书脊阴影，增强实体触感
- **不打扰**：不弹窗、不推送、不催促，让用户按照自己的节奏使用

### 无障碍

- 所有交互元素支持键盘操作
- 支持 `prefers-reduced-motion: reduce` 减少动效
- 日夜模式自适应（`data-theme="night"`）
- 图片加载失败静默降级

# CHANGELOG -- 森林疗愈室

---

## V1.0 (2026-07-03) -- 首次发布

### 新增功能

#### 导航框架

- 左侧 220px 固定侧边栏导航，包含 4 个模块入口（感恩日记/呼吸引导/冥想空间/留言板）
- 导航项包含中文标签、英文副标题、细线 SVG 图标
- 选中态左侧绿色竖线指示器，通过 Framer Motion layoutId 实现平滑过渡动画
- 模块内容区切换使用淡入淡出动画（AnimatePresence）
- 移动端（<768px）自动切换为顶部固定栏 + 汉堡菜单 + 下拉导航面板
- 默认欢迎卡片，引导用户选择模块
- "回到主站"按钮，通过 React Router Link 跳转

#### 背景层

- 固定全屏森林背景图，带渐变暗角（上下加深）
- 三片树叶 CSS 微风摇曳动画（三层独立 keyframes，不同速度/方向/延迟）
- 半透明深色遮罩层，确保前景内容可读性

#### 感恩日记

- 实体绘本式三页交互模型（封面/目录/日记）
- 封面页：手绘水彩风格背景图 + 柔光遮罩 + 漂浮光斑动画 + 中英文标题
- 点击封面触发 3D 翻书动画（rotateY 变换 + 书脊阴影）
- 目录页：12 个月度篇章 4 列网格（移动端 3 列），当前月份高亮边框 + 发光效果
- 已有日记的月份右上角显示绿色圆点标记
- 月份卡片 hover 上浮动画
- 日记页：月度引导语引用块（带左侧绿色边框）
- 天气选择器：4 种 SVG 细线图标（晴天/多云/雨天/夜晚）
- 心情色环：6 色可选（开心/平静/冷静/温柔/感性/平淡），选中放大 + 白边 + 发光
- 书写文本域：纸张质感背景（横线纹理 + 纤维噪点 + 水彩晕染）
- 便签贴纸风格保存按钮（旋转 -2deg），保存后显示反馈文案 2 秒
- 本月历史记录列表，带时间轴竖线 + 圆点，支持删除
- 草稿自动保存到 localStorage（天气/心情/内容实时持久化）
- 右下角固定"合上书"按钮，返回封面
- 兼容旧版 emoji 格式 mood 数据

#### 呼吸引导

- 三种呼吸模式：方形呼吸(4-4-4-4)、478 呼吸法(4-7-8)、能量呼吸法(3-1-2)
- 三个情绪状态选择按钮（焦虑/压力大、疲惫/失眠、能量不足），点击切换对应模式
- rAF（requestAnimationFrame）驱动的动画引擎：
  - 光球缩放：吸气 0.76 -> 1.14，呼气 1.14 -> 0.76
  - SVG 圆弧进度：吸气填充，屏息保持满，呼气消退
  - 中心数字倒计时（等宽字体）
- 核心 Bug 修复：`lastSecond = -1` 强制首帧更新，解决阶段切换时倒计时数字跳跃问题
- 双 Effect 架构：Effect A 处理阶段重置（状态驱动），Effect B 处理 rAF 动画（时间驱动）
- 节奏指示器：圆点 + 阶段标签，当前阶段高亮，支持 3/4 阶段自适应间距
- 循环计数显示
- 开始/暂停/继续三态按钮 + 重置按钮
- 暂停后从断点续接，不重置循环

### 技术实现

- 技术栈：React 18 + TypeScript + Framer Motion + Web Audio API
- 样式方案：组件级内联 `<style>` 标签 + CSS Variables 主题变量
- 数据持久化：localStorage（`gratitude_entries` / `gratitude_drafts` / `meditation_total`）
- 动画方案：Framer Motion（布局动画、翻书、涟漪）+ rAF（呼吸引导高频动画）+ CSS keyframes（树叶、光斑）
- 响应式布局：桌面端左侧导航 + 右侧内容区，移动端顶部导航 + 下拉面板
- 日夜模式：所有组件支持 `data-theme="night"` 暗色适配
- 无障碍：支持 `prefers-reduced-motion: reduce` 减少动效，键盘可操作
- 性能优化：动画仅使用 transform/opacity/stroke-dashoffset（GPU 友好），图片懒加载

### 设计风格

- 日系文艺风：森林自然意象，低饱和绿色系主色调
- 纸张质感：内页模拟手造纸纹理（横线 + 纤维噪点），暖白/奶油色底
- 毛玻璃层次：导航栏和内容卡片使用 backdrop-filter 毛玻璃效果
- 水彩晕染：多层径向渐变叠加，模拟森林光晕透入纸面
- 细线 SVG 图标：统一克制视觉风格
- 呼吸感动画：0.3-0.6s easeOut 缓动，营造"缓慢而温柔"的节奏
- 3D 翻书隐喻：实体触感交互

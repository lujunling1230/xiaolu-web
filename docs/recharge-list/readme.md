# 回血清单 (Recharge Station) -- README

## 产品简介

回血清单是妙妙工具箱「心」维度下的情绪修复工具，定位为"微型能量站"。核心隐喻是"给生命充电"：100件滋养身心的小事被设计为垂直排列的能量节点，每一件是一个胶囊电池，用户点击完成即相当于接通电源，补充生命能量。

列表底部隐藏着第101个电源，一个哲学节点："允许自己永远充不满电" -- 传递自我接纳的价值观。右下角毛玻璃计数器记录本周回血次数，每周一自动重置，帮助用户建立可持续的自我关怀节奏。

路由：`/toolbox/recharge`

## 功能列表

| 功能模块 | 描述 |
|----------|------|
| 能量节点流 | 100件回血小事的垂直列表，每件包含图标、文字、电池指示器，点击完成/取消 |
| 充能反馈 | 点击完成时触发8个绿色粒子爆发动画，同时播放电流声（Web Audio API） |
| 胶囊电池动画 | 每个节点右侧的电池指示器，完成时绿色渐变填充，带外发光 |
| 连接线流动 | 节点间垂直连接线带流动小点，hover时金色，完成时绿色 |
| 本周回血计数 | 右下角毛玻璃固定卡片，显示本周累计完成次数，每周一自动归零 |
| 第101个隐藏电源 | 列表底部极小的隐藏按钮，点击弹出毛玻璃哲学弹窗 |
| 小猫SVG动画 | 底部装饰小猫，睡眠/睁眼/哈欠三态，与隐藏电源联动 |
| 心电图背景 | SVG纹理背景，页面加载时呈现波动动画 |
| 数据持久化 | localStorage存储完成状态和计数，支持页面刷新恢复 |

## 技术栈

| 技术 | 用途 |
|------|------|
| React 18+ | 前端框架 |
| TypeScript | 类型安全 |
| Framer Motion | 动画引擎（节点入场、粒子爆发、弹窗过渡、计数器弹性） |
| Web Audio API | 电流声合成（sawtooth波形，800Hz->400Hz，0.15s） |
| SVG | 心电图背景纹理、小猫角色动画 |
| CSS backdrop-filter | 毛玻璃效果（胶囊节点、计数器、弹窗） |
| localStorage | 客户端数据持久化 |

## 本地存储说明

| 键名 | 类型 | 说明 |
|------|------|------|
| `recharge_count` | string(Number) | 本周完成计数 |
| `recharge_done_ids` | string(JSON Array) | 已完成节点ID数组，如 `[1, 5, 23]` |
| `recharge_count_ts` | string(Number) | 最近一次计数时间戳，用于判断是否跨周 |

**周重置逻辑**：每次加载时读取 `recharge_count_ts`，若早于本周一00:00则将计数归零。完成状态（`recharge_done_ids`）不随周重置清除。

**容错策略**：所有 localStorage 读写均包裹 `try-catch`，存储失败或数据损坏时静默降级为内存数据。

## 设计理念

### 能量隐喻体系

整个产品围绕"充电"隐喻构建了一致的符号系统：

- **节点 = 电源接口**：每件小事是一个可接入的电源
- **电池 = 能量状态**：完成即充满，未完成即待充
- **连接线 = 电路**：节点间以电路线串联，传递能量
- **心电图 = 生命体征**：背景纹理暗示"检测生命能量"
- **第101电源 = 哲学收纳**：接受能量不总是满格，自我接纳

### 日系文艺风格

- 暖白底色（`#FDFBF7`）搭配浅灰绿文字（`#5a5048`），营造温暖、柔和的阅读氛围
- Noto Serif SC 衬线字体用于标题，传递文艺气质
- 半透明毛玻璃材质（`backdrop-filter: blur`）赋予轻盈通透的质感
- 图标池混合emoji与中文字符，保持视觉趣味性

### 反馈层次

遵循"即时、多层、有节制"的反馈原则：

1. **视觉层**：电池填充 + 粒子爆发 + 边框变色
2. **听觉层**：电流声（0.15s，短促不扰人）
3. **数据层**：计数+1 + 弹性动画

三层反馈在点击瞬间同时触发，形成强烈的"完成感"。

### 情绪节奏

- 顶部标题区留白充足，给用户"进入"的空间
- 节点列表密集但不拥挤，滚动浏览有"探索"的节奏感
- 底部隐藏电源+小猫，为浏览终点提供惊喜和柔软收尾
- 计数器固定在右下角，不干扰主内容但随时可见

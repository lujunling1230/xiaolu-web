# 时光博物馆 (Museum of Memories) 更新日志

---

## V1.0 -- 首次发布 (2026-07-03)

### 产品概要

时光博物馆 V1.0 正式发布。作为妙妙工具箱「魂」维度的核心产品，本版本实现了两个主题展厅的完整功能，覆盖 16 项功能需求，奠定了复古胶片风格的设计语言基调。

---

### 新增功能

#### 展厅一：时代回响 (Hall of Echoes)

- **三子类目 Tab 切换**：实现「时代BGM」「时代荧幕」「时代网络」三个子类目之间的平滑切换，使用 Framer Motion 动画过渡
- **时代BGM 展品**：内置 12 首默认经典歌曲，涵盖 2000-2015 年华语及流行音乐
- **时代荧幕展品**：内置 11 部默认经典影视作品，包含国产剧、动画、电影
- **时代网络展品**：内置 3 条默认互联网文化记忆条目，涵盖 Flash 小游戏、贴吧、论坛
- **复古报纸风格卡片**：每张卡片采用深褐底色 + 暗金边框 + 衬线字体的报纸风格渲染
- **迷你音乐播放器**：BGM 类目支持网易云音乐 iframe 嵌入式播放，点击即播

#### 展厅二：荣耀之路 (Hall of Glory)

- **垂直时间轴布局**：荣誉条目按年份纵向排列，支持时间轴视觉引导
- **5 条默认荣誉**：内置国家励志奖学金、优秀志愿者、数据库系统工程师证书、网络安全证书、英语四级证书
- **复盘反思展开/折叠**：每条荣誉支持展开查看详细复盘内容，折叠/展开使用 Framer Motion 高度动画
- **高光时刻光晕**：重要荣誉条目附带金色光晕装饰效果

#### 全局功能

- **管理员认证系统**：实现 `useAdminGuard` Hook，密码 `"ling"` 校验，未认证隐藏所有编辑操作
- **展品 CRUD 操作**：管理员可对任意展品执行新增、编辑、删除操作
- **图片上传功能**：支持 JPG/PNG/GIF/WEBP 格式图片上传，通过 `FileReader` 转为 Base64 存储
- **确认删除对话框**：删除操作前弹出 `ConfirmDialog` 确认框，防止误操作
- **数据持久化**：所有展品数据存储于浏览器 localStorage，分 4 个 Key 独立管理
- **数据迁移兼容**：页面加载时自动检测旧数据，补全缺失的 `imageUrl` 字段

#### 视觉特效

- **尘埃飘落粒子动画**：全屏 CSS 粒子动画，模拟老照片质感
- **胶片显影入场动画**：页面加载时内容从模糊到清晰的渐进式呈现

### 技术实现

#### 前端架构

- **框架**：React 18 + TypeScript 5
- **动画引擎**：Framer Motion 10，覆盖卡片过渡、复盘展开、页面入场等全部动画场景
- **ID 生成**：uuid v9，确保每条展品唯一标识

#### 组件化设计

- **MuseumPage**：主容器组件，约 2400 行，管理全局状态、数据流与 CRUD 逻辑
- **VintageCard**：展厅一复古卡片组件，负责报纸风格渲染与音乐播放器入口
- **HonorItem**：展厅二荣誉条目组件，负责时间轴布局与复盘折叠
- **CardForm**：通用表单组件，支持新增/编辑两种模式，含图片上传与预览
- **MiniPlayer**：迷你音乐播放器组件，嵌入网易云音乐 iframe
- **ConfirmDialog**：确认对话框组件，用于删除二次确认
- **useAdminGuard**：自定义 Hook，封装管理员认证状态管理

#### 数据模型

- **VintageCard 接口**：定义展厅一展品数据结构（id, year, title, description, imageUrl, musicUrl?）
- **HonorItem 接口**：定义展厅二荣誉数据结构（id, year, title, description, imageUrl, reflection）
- **localStorage 4 Key 结构**：`museum_bgms`、`museum_tvs`、`museum_nets`、`museum_honors` 独立管理

#### 数据兼容

- 实现旧数据自动迁移逻辑：检测 `museum_bgms`、`museum_tvs`、`museum_nets`、`museum_honors` 中缺失的 `imageUrl` 字段并自动补全为空字符串

### 设计风格

#### 色彩体系

- 主背景：`#5c4033`（VINTAGE_DEEP 深褐色）
- 暗金边框：`#b08d57`（GOLD 暗金色）
- 辅助色：`#8B6B4F`（VINTAGE_BROWN 复古棕）
- 卡片底色：`#3a2a1a`（复古报纸深褐）
- 文字主色：`#f5e6d3`（泛黄纸张色）

#### 字体

- 主字体：Noto Serif SC（思源宋体），Google Fonts 加载
- 备选：Georgia, 'Times New Roman', serif

#### 动画语言

- 卡片入场：淡入 + 轻微上移，Framer Motion `fadeInUp` 变体
- Tab 切换：列表内容交叉淡入淡出
- 复盘展开：高度从 0 到 auto 的弹性动画
- 页面入场：胶片显影效果（模糊到清晰）
- 粒子背景：CSS keyframes 尘埃飘落循环动画

---

### 已知限制

- 管理员密码 `"ling"` 为明文硬编码，仅适用于个人站点场景，不具备安全防护能力
- 数据仅存储在本地浏览器，不支持多设备同步
- 图片以 Base64 存储，受 localStorage 单 Key 5MB 容量限制
- 音乐播放依赖网易云音乐外链，若外链失效则无法播放

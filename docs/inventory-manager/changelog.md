# 物资管家 (Inventory Prophet) -- CHANGELOG

## V1.0 (2026-07-03) -- 首次发布

### 新增功能

- 入库登记表单：支持物品名称、数量、单位（瓶/盒/袋/罐/箱/个/支）、到期日、存放位置（冰箱/浴室/储物间/厨房/客厅/卧室）五个字段的录入
- 入库表单校验：名称非空、数量最低为 1、到期日必填，校验不通过时提交按钮禁用
- 连续入库优化：入库成功后仅清空物品名称，保留数量、单位、位置，方便连续录入同类物品
- 状态仪表盘：三态胶囊按钮（已过期/临期30天/库存充足），实时统计各状态物品数量，点击可切换筛选
- 胶囊反向筛选：再次点击同一胶囊取消筛选，显示全部物品；筛选激活时出现"显示全部"快捷按钮
- 库存列表表格：表格展示全部物品，包含物品名称、数量+单位、到期日、存放位置、操作按钮五列
- 状态视觉标识：表格行左侧 4px 彩色竖条（红/橙/绿），到期日文字颜色与状态对应，过期物品显示红色"已过期"徽章
- 相对时间文案：到期日列显示友好提示（"已过期 N 天"/"今天到期"/"还剩 N 天"）
- 编辑弹窗：模态编辑弹窗，预填充当前值，支持修改物品全部字段，点击遮罩层关闭
- 删除淡出动画：删除操作触发 300ms 透明度过渡动画，动画结束后从 DOM 移除
- 管理员权限保护：编辑和删除操作需通过管理员密码验证，防止误操作
- 智慧规划面板：赏味期限面板自动识别最早过期物品，提供差异化消耗建议
- 赏味期限文案分支：临期物品（<=30天）显示"赏味期限将至，宜趁鲜启用。"及"立即消耗"按钮；充足物品显示"余量丰盈，且容它静候时光。"及"留着以后"按钮
- 空状态引导：无物品时入库表单及赏味期限面板均显示引导文案
- 路由别名：`/toolbox/supplies` 作为 `/toolbox/inventory` 的别名路由
- 返回导航透传：顶栏返回链接支持透传 `from=full` 查询参数

### 技术实现

- 基于 React 18 + TypeScript 构建，使用函数组件 + Hooks 模式
- 数据持久化采用 localStorage，key 为 `inventory_items`，useEffect 监听 items 变更自动同步
- ID 生成优先使用 `crypto.randomUUID()`，不兼容环境回退为 `时间戳-随机字符串` 方案
- 到期日比较基于当天零点（`setHours(0,0,0,0)`），避免跨天比较偏差
- 状态计算使用 `useMemo` 缓存，避免不必要的重计算
- 管理员权限通过 `useAdminGuard` Hook 统一管理，编辑/删除操作前调用 `verifyAdmin` 进行验证
- 删除动画通过 `deletingIds` Set 集合管理，CSS transition 实现 300ms 淡出
- 响应式布局：桌面端（>=1024px）左右双栏，左侧 340px 固定 + 右侧弹性；移动端单栏垂直布局
- 左侧面板在桌面端使用 `sticky` 定位，滚动时保持可见
- 表单校验实时计算，按钮 disabled 状态与 `canAdd` 变量绑定
- 编辑弹窗通过遮罩层点击关闭，弹窗内部点击阻止冒泡
- 数据写入异常（隐私模式等）静默捕获，不影响正常使用

### 设计风格

- 主色调：鼠尾草绿 #5d8a6a（按钮、链接、焦点状态）
- 状态颜色：红色 #E53935（过期）、橙色 #FF9800（临期）、绿色 #81C784（充足）
- 布局风格：SaaS 紧凑型白色卡片 + 浅灰背景 (#f9fafb)
- 胶囊按钮：圆角全圆（rounded-full），激活态彩色背景，空闲态白色背景
- 表格样式：白色背景圆角卡片，边框线 #e5e7eb，表头灰色小字
- 表格行左侧 4px 彩色状态竖条（border-left）
- 编辑弹窗：圆角 16px 白色卡片，半透明黑色遮罩 + 模糊效果
- 浮动管理员入口：圆形毛玻璃按钮，右下角固定定位
- 徽章样式：过期物品红色药丸徽章（background: rgba(229,57,53,0.1)）
- 赏味期限面板：居中对齐布局，状态对应颜色文案，圆角药丸按钮

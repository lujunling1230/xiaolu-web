# 解压馆 (Stress Relief Room) -- README

## 产品简介

解压馆是妙妙工具箱「感」维度下的触觉发泄工具，聚焦于感官层面的即时解压。提供三款马卡龙色系的小游戏，零门槛即点即玩，无需教程，无需注册。

三款游戏分别对应不同的感官通道：捏泡泡（触觉+听觉）、禅意切割（视觉+动觉）、重力涂鸦（动觉+视觉）。所有内容可无限循环重生，无结束条件，无压力。

路由：`/toolbox/games`

## 功能列表

| 功能模块 | 描述 |
|----------|------|
| 游戏选择卡片 | 三张横向卡片，展示游戏图标、名称、描述，点击进入 |
| 无限捏泡泡 | 6x6 SVG气泡网格，点击爆裂+星星粒子+啵声，600ms重生换色 |
| 禅意切割 | 5个彩色方块，PointerEvent拖拽划线切割，clipPath裂成两半飞散，完成后重生 |
| 重力涂鸦 | Canvas画线生成圆点，受重力下坠堆积，高度图列检测，清空按钮 |
| 游戏切换 | 游戏内"换一个"按钮返回卡片选择页，带缩放过渡动画 |
| 马卡龙色板 | 6色全局色板，气泡/方块/圆点/粒子随机取色 |

## 本地存储说明

解压馆不进行任何持久化存储。所有游戏状态（气泡、方块、圆点、切割线）均为会话级内存数据，页面刷新或关闭后全部重置。

设计意图：解压是即时的、不追求积累的行为。零存储降低用户心理负担，每次进入都是全新的开始。

## 设计理念

### 感官通道理论

三款游戏的设计遵循"多感官通道覆盖"原则，确保不同解压偏好的用户都能找到适合自己的方式：

| 游戏 | 触觉 | 听觉 | 视觉 | 动觉 |
|------|------|------|------|------|
| 捏泡泡 | 主 | 辅 | 辅 | - |
| 禅意切割 | 辅 | - | 主 | 主 |
| 重力涂鸦 | - | - | 主 | 主 |

### 马卡龙美学

- 粉色渐变背景（`radial-gradient`），营造柔和甜美的氛围
- 6色马卡龙色板（浅粉、浅蓝、浅绿、浅紫、浅黄、肉粉），每种颜色饱和度低、明度高
- 圆角卡片（18px）+ 内阴影高光，模拟实体甜品的质感
- 柔和阴影（`box-shadow: 0 10px 30px -14px`），避免生硬投影

### 无压力哲学

- **无教程**：所有交互直觉化，点击即玩
- **无分数**：不追踪成绩，不比较，不评判
- **无结束**：内容无限循环重生，没有"通关"压力
- **无存储**：不积累数据，每次进入都是新开始
- **可切换**：随时"换一个"，不锁定用户

### 物理交互深度

- 捏泡泡：爆裂+粒子+音效，模拟真实捏泡泡的满足感
- 禅意切割：拖拽划线+clipPath分裂+飞散旋转，模拟切割的视觉冲击
- 重力涂鸦：重力模拟+高度图堆积+插值画线，模拟物理世界的因果关系
